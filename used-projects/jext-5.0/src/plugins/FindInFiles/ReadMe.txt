FindInFiles is a Jext plugin that lets you through files. Jext is an open source editor that can be downloaded at http://jext.org. This plugin enables you to search a directory structure for a pattern. It supports regular expressions and file filters. Multi-clicking on search results opens the file for editing.

Installation: Unzip this file FindInFiles.zip to jextHome/bin/plugins directory, where 'jextHome' is the Jext install directory. 

Running: After Jext is restarted, select the menu option 'Find in Files Window' in the 'Plugins' menu. This will turn on/off the Find in Files window in the tabbed option pane. Choose the 'Find In Files' tab, enter search criteria, and click searching. Multi-clicking on the search results will open the file.
